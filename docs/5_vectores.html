5. Vectores: La estructura fundamental

Los vectores son **secuencias o conjuntos de elementos del mismo tipo de datos**. En R, la unidad básica de almacenamiento es el vector; incluso un número aislado se trata como un vector de longitud uno. Se crean utilizando la función c() (combine).

# --- Creación y manipulación ---
x <- c(4.1, 5, 6.35)
x                          # Imprime el vector
## [1] 4.10 5.00 6.35
x <- c(7, x, 2)            # Añade elementos al inicio y final
x                          # Resultado: 7, 4.1, 5, 6.35, 2
## [1] 7.00 4.10 5.00 6.35 2.00
length(x) # Cantidad de elementos 
## [1] 5
mode(x) # Tipo de datos: "numeric"
## [1] "numeric"

5.1. Coerción de tipos

Los vectores en R deben contener elementos del mismo tipo de datos. Si intentamos combinar tipos distintos en un mismo vector, R aplicará la coerción, convirtiendo automáticamente todos los elementos al tipo de datos más flexible (o “general”).

# Definimos un vector de caracteres y uno numérico
y <- c("Hola", "Adios")
x <- c(4.1, 5, 6.35)

# Intentamos mezclar textos, números y valores lógicos
y_mezclado <- c(y, x, TRUE)

# Verificamos el resultado
y_mezclado          # Todo se ha convertido a texto ("character")
## [1] "Hola"  "Adios" "4.1"   "5"     "6.35"  "TRUE"
mode(y_mezclado)    # El vector es ahora de tipo "character"
## [1] "character"
length(y_mezclado)  # 7 elementos en total (2 textos + 3 números + 1 lógico)
## [1] 6

5.2. Acceder a elementos de un vector (Indexación)

En R, los vectores son indexados empezando por 1 (no por 0 como en otros lenguajes como Python). Utilizamos corchetes [] para extraer valores específicos.

mi_vector <- c(8, 9, 3, 4, 5)

# Acceder al segundo elemento
elemento <- mi_vector[2]
print(elemento)
## [1] 9
# Verificar el tipo de datos del vector
mode(mi_vector)
## [1] "numeric"

5.3. Generación de secuencias

Existen diversas formas de crear vectores numéricos ordenados sin escribir cada elemento manualmente:

# El operador ':' genera una secuencia de enteros consecutivos
# NOTA: Los paréntesis alrededor de la asignación guardan el valor y lo imprimen a la vez
(x <- 1:5) 
## [1] 1 2 3 4 5
# Función seq(): Permite definir el incremento (by)
seq(from = 1, to = 5, by = 0.5)
## [1] 1.0 1.5 2.0 2.5 3.0 3.5 4.0 4.5 5.0
# Función seq(): Permite definir el número total de elementos (length.out)
seq(from = 1, to = 5, length.out = 6)
## [1] 1.0 1.8 2.6 3.4 4.2 5.0
# Función sample(): Genera secuencias aleatorias
# 'replace = TRUE' permite que los números se repitan
sample(1:6, size = 10, replace = TRUE) 
##  [1] 4 6 3 5 5 1 3 1 2 2

Notas:

  • Paréntesis de impresión: Cuando haces (x <- 1:5), le ahorras a R una línea de código al realizar la asignación y la visualización en un solo paso. Es un truco muy común en sesiones interactivas.
Método tradicional:
  x < - 1:5
  print (x)

Método con paréntesis:
 (x < - 1:5)

5.4. Operaciones vectorizadas

La gran ventaja de los vectores en R es que permiten aplicar funciones u operaciones matemáticas a todos los elementos simultáneamente, sin necesidad de escribir bucles.

x <- c(1, 2, 3, 4)

# Multiplicar todos los elementos por 2 de golpe
x * 2
## [1] 2 4 6 8
# Elevar cada elemento al cuadrado
x ^ 2
## [1]  1  4  9 16
# Sumar dos vectores (se suman elemento a elemento)
y <- c(10, 20, 30, 40)
x + y
## [1] 11 22 33 44

5.5. Filtrado lógico (Subsetting)

Podemos extraer elementos de un vector basándonos en condiciones. Esto es extremadamente útil para limpiar datos.

datos <- c(10, 25, 5, 40, 15)

# ¿Qué elementos son mayores que 20?
datos > 20
## [1] FALSE  TRUE FALSE  TRUE FALSE
# Extraer solo los elementos que cumplen la condición
datos[datos > 20]
## [1] 25 40
# Extraer elementos que son iguales a 10 O mayores que 30
datos[datos == 10 | datos > 30]
## [1] 10 40
