Operadores en R

Aritméticos

Operador Descripción
+ Suma
- Resta
* Multiplicación
/ División
^ Potencia
%/% División entera

Ejemplos:

# Definición de variables
a <- 10
b <- 3

# Resultados de las operaciones
resultado_suma    <- a + b
resultado_resta   <- a - b
resultado_multi   <- a * b
resultado_div     <- a / b
resultado_pot     <- a ^ b
resultado_entera  <- a %/% b

# Imprimir solo los resultados
print(resultado_suma)
## [1] 13
print(resultado_resta)
## [1] 7
print(resultado_multi)
## [1] 30
print(resultado_div)
## [1] 3.333333
print(resultado_pot)
## [1] 1000
print(resultado_entera)
## [1] 3

Comparativos

Operador Descripción
== Igualdad
!= Diferente de
< Menor que
> Mayor que
<= Menor o igual
>= Mayor o igual

Ejemplo:

# Definimos valores para comparar
x <- 10
y <- 5

# 1. Igualdad (==)
print(x == y)      # ¿Es 10 igual a 5? (FALSE)
## [1] FALSE
# 2. Diferente de (!=)
print(x != y)      # ¿Es 10 diferente de 5? (TRUE)
## [1] TRUE
# 3. Menor que (<)
print(x < y)       # ¿Es 10 menor que 5? (FALSE)
## [1] FALSE
# 4. Mayor que (>)
print(x > y)       # ¿Es 10 mayor que 5? (TRUE)
## [1] TRUE
# 5. Menor o igual (<=)
print(x <= 10)     # ¿Es 10 menor o igual a 10? (TRUE)
## [1] TRUE
# 6. Mayor o igual (>=)
print(y >= 15)     # ¿Es 5 mayor o igual a 15? (FALSE)
## [1] FALSE

Lógicos

Operador Descripción
& Y lógico
| O lógico
! No lógico

Ejemplos:

x <- 10
y <- 5

# 1. Y lógico (&)
# ¿Es x mayor que 5 Y es y menor que 10?
print(x > 5 & y < 10) # TRUE porque ambas se cumplen
## [1] TRUE
# 2. O lógico (|)
# ¿Es x igual a 100 O es y igual a 5?
print(x == 100 | y == 5) # TRUE porque al menos una se cumple
## [1] TRUE
# 3. NO lógico (!)
# ¿Es x diferente de 10? (No, es igual, así que dará FALSE)
print(!(x == 10))
## [1] FALSE
